# Blinkit_Sales_Performance_Dashboard
Developed an interactive Power BI dashboard for Blinkit sales analysis to track key business metrics such as total sales, average sales, item performance, outlet size, outlet location, and customer ratings. 
Created dynamic visualizations, KPIs, filters, and data-driven insights using Power BI, DAX measures, and data modeling techniques to support business decision-making and performance analysis.
## Key Metrics
- Total Sales: $1.2M+
- Average Sales
- Item Performance
- Outlet Performance
- Customer Ratings

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Data Modeling

## Key Insights
- Analyzed sales across outlet types and locations.
- Created dynamic KPIs and interactive visualizations.
- Enabled data-driven decision-making through dashboard reporting.
